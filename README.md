###Getting Started###

After cloning the app, run
```bash
npm install
```

######To build the app, run######
```bash
npm run build
```

######To start the app, run######
```bash
npm run start
```
#####You can now view the site at localhost:3000#####


###Testing###

######To launch end to end tests, make sure to start the app and run######
```bash
npm run e2e-setup
nightwatch
```

######To launch unit tests, run######
```bash
npm run test
```
